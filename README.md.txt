# ProductService.API
This is the NET Core API is developed using .net Core 6 version.
Patterns implemented are:
- CRQS
- Mediator
- Repository
- Dependency Injection

The architecture implemented is Domain Driven Design.

Persistence data implemented is Entity Framework Core using In Mememory Database approach.

External service used for this demo is
https://627d679058ff74eaeea26027.mockapi.io/api/v1/productDiscount

API documentation is using Swagger tool.

# Installation process
1. Create folder named Login in C drive to see log files every time you make a request.
   The result should looks like this C:\Log

2. Open project using Visual Studio and make sure you download all nuget packages needed.

3. Start.
